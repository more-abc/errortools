# Installation

## Requirements

- Python 3.8 or higher
- No external dependencies for core functionality

## Install from PyPI

```bash
pip install errortools
```

## Install from source

```bash
git clone https://github.com/more-abc/errortools.git
cd errortools
pip install -e .
```

## Verify installation

```python
import errortools
print(errortools.__version__)
```
